# Rogs Text Editor

Many centuries ago I did one in Delphi 6 (!) then one day staring at Star Trek on the goggle box I thought "wonder if I can write one in the modern age using C#?"

Done for fun, still in development, how hard can it be?


**Features:**
- bold/italic/underline
- font size
- font colour
- font type
- supports text and RTF
- loads
- prints
